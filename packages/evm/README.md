## Install

You will need a copy of [Foundry](https://getfoundry.sh/) installed before proceeding.

### Setup

```sh
forge install
pnpm install
```

### Build

```sh
forge build
```

### Lint

```sh
pnpm lint
```

### Format

```sh
pnpm format
```

### Test

```sh
forge test
```

### Gas Snapshot

```sh
forge snapshot
```
